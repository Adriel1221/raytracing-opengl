### RayTracing OpenGL

Implemented with OpenGL fragment shaders.

Scene setup in main.cpp source file.

#### Controls:

Rotate camera with mouse

Movement:

- WASD
- Space - up
- Ctrl - down
- Shift (hold) - boost
- Alt (hold) - slowdown

#### Requirements

* CMake (>= 3.0.2)
* GPU with OpenGL (>= 3.1) support
* GLFW, should be automatically found by CMake.
